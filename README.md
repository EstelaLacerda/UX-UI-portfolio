## Project Structure

```
ux-ui-portfolio/
├── public/                 # Static assets (images, CV PDF)
├── src/
│   ├── components/         # Reusable React components
│   │   ├── Header.tsx      # Main header and hero section
│   │   ├── ExpertiseSection.tsx # Skills and expertise display
│   │   ├── ExperienceSection.tsx# Professional experience timeline
│   │   ├── WorkSection.tsx # Projects grid showcase
│   │   └── ProjectModal.tsx# Detailed view for each project
│   ├── contexts/           # React Context providers
│   │   └── LanguageContext.tsx # i18n logic and state
│   ├── locales/            # Translation dictionaries
│   │   ├── en.ts           # English strings
│   │   └── pt.ts           # Portuguese strings
│   ├── types/              # TypeScript type definitions
│   │   └── project.ts      # Project interface schema
│   ├── App.tsx             # Main application layout
│   ├── index.css           # Global Tailwind utilities
│   └── main.tsx            # React DOM entry point
├── package.json            # Project dependencies
├── tailwind.config.js      # Custom theme and fonts
├── tsconfig.json           # TypeScript configuration
└── vite.config.ts          # Vite bundler settings
```

## How to Run

### <img src="https://skillicons.dev/icons?i=github" height="20" style="vertical-align: middle;" /> 1. Clone the repository

```bash
git clone https://github.com/EstelaLacerda/UX-UI-portfolio.git
cd UX-UI-portfolio
```

### 📦 2. Install dependencies

```bash
npm install
```

### ▶️ 3. Start the application

```bash
npm run dev
```

The app will be available at **http://localhost:5173**.

## Estrutura do Projeto

```
ux-ui-portfolio/
├── public/                 # Arquivos estáticos (imagens, PDF do currículo)
├── src/
│   ├── components/         # Componentes React reutilizáveis
│   │   ├── Header.tsx      # Cabeçalho principal e seção hero
│   │   ├── ExpertiseSection.tsx # Exibição de habilidades e especialidades
│   │   ├── ExperienceSection.tsx# Linha do tempo de experiência profissional
│   │   ├── WorkSection.tsx # Grade de exibição de projetos
│   │   └── ProjectModal.tsx# Visualização detalhada para cada projeto
│   ├── contexts/           # Provedores de Contexto do React
│   │   └── LanguageContext.tsx # Lógica e estado de internacionalização (i18n)
│   ├── locales/            # Dicionários de tradução
│   │   ├── en.ts           # Textos em Inglês
│   │   └── pt.ts           # Textos em Português
│   ├── types/              # Definições de tipos do TypeScript
│   │   └── project.ts      # Esquema da interface de Projetos
│   ├── App.tsx             # Layout principal da aplicação
│   ├── index.css           # Utilitários globais do Tailwind
│   └── main.tsx            # Ponto de entrada do React DOM
├── package.json            # Dependências do projeto
├── tailwind.config.js      # Configurações de tema e fontes
├── tsconfig.json           # Configuração do compilador TypeScript
└── vite.config.ts          # Configuração do Vite
```
